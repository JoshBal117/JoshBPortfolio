Projects

Selected work that showcases my skills and interests.

Indie Tactical Roguelike

Dungeon Delver

A tactical roguelike dungeon crawler built with React and TypeScript. I implemented a custom turn-based combat engine, enemy AI behaviors, XP and leveling logic, and an event-driven state system using Zustand.

The project also features original 32×32 pixel-art sprites, animation integration, and a deployment pipeline via GitHub Pages. I am actively prototyping a Unity-based port for a future Steam release.

Prototype Storefront

Crit & Counter

A prototype e-commerce storefront focused on collectible game merchandise and tabletop accessories. Built with React and Node.js, it includes product listing views, early inventory logic, and a responsive layout tuned for both desktop and mobile users.

The project is designed as a foundation for a future production-ready storefront with full cart, checkout, and account features.
